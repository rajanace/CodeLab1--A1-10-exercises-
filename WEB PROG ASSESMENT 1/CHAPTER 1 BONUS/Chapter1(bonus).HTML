Essential CSS Properties

color

Sets the text color.

p { color: blue; }

font-size

Defines the size of text.

h1 { font-size: 24px; }

background-color

Sets the background color of an element.

body { background-color: lightgray; }

margin

Defines the space around an element.

div { margin: 10px; }

padding

Defines the space inside an element.

div { padding: 15px; }
